DOCUMENT    -> BLOCKLIST
BLOCKLIST   -> BLOCK
BLOCKLIST   -> BLOCK BLOCKLIST
BLOCK       -> HEADING
BLOCK       -> PARAGRAPH

HEADING     -> # TEXT \n
PARAGRAPH   -> TEXT \n

TEXT        -> ELEMENT
TEXT        -> ELEMENT TEXT
ELEMENT     -> BOLD
ELEMENT     -> ITALICS
ELEMENT     -> PLAINTEXT

BOLD        -> ** PLAINTEXT **
ITALICS     -> * PLAINTEXT *
PLAINTEXT   -> characters
PLAINTEXT   -> characters PLAINTEXT
